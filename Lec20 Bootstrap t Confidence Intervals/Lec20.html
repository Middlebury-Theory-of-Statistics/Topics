Today's Population

Let's take a random sample of 10% of OkCupid users' heights and compute summary statistics:

heights <- profiles$height
n <- length(heights)
xbar <- mean(heights)
s <- sd(heights)
n xbar s
5994 68.244 3.95

Bootstrap Resample T*

N <- 10^4
t_star <- rep(0, N)

for(i in 1:N){
  resample <- sample(heights, size=n, replace=TRUE)
  t_star[i] <- (mean(resample)-xbar)/(sd(resample)/sqrt(n))
}
