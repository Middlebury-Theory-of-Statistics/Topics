Example 1: We know result

Let the population distribution be Normal\((\mu=23, SD=7)\)…

Example 1: We know result

from which you draw a sample of size \(n=50\). We have \(\overline{x}\) = 22.181 and:

Example 1: We know result

We know the sampling distribution of \(\overline{X}\) is Normal\((\mu=23, SD=7/\sqrt{50}=0.99)\):

Example 1: We know result

The bootstrap distribution of \(\overline{X}\) is similar in shape and spread as the true sampling distribution, but it is not centered at \(\mu=23\), rather at \(\overline{x}\) = 22.181.

Standard Error

The bootstrap standard error (i.e. approximation) is:

sd(boot_sample_means)
## [1] 1.042678

The true standard error is: \[ \text{SE} = \frac{\sigma}{\sqrt{n}} = \frac{7}{\sqrt{50}} =0.99 \]

Example 2: We don't know result

Now say we want to know the sampling distribution of a wacky statistic:

  • \(T(\vec{X}) = \frac{\max(\vec{X}) - \min(\vec{X})}{\text{median}(\vec{X})}\)
  • with \(X_i \sim\) Normal\((\mu=27, \sigma=7)\) IID RV
  • and \(n=5000\)

Example 2: We don't know result

There is no result for what the sampling distribution of \(T(\overline{X})\) is, or at least, it's hard to derive. So we create the bootstrap distribution:

Example 2: We don't know result

n <- 5000
x <- rnorm(n, mean=23, sd=7)

boot_sample_WTF <- rep(0, 10000)

for(i in 1:10000){
  boot_resample <- sample(x, replace = TRUE)
  
  boot_sample_WTF[i] <- 
    (max(boot_resample) - min(boot_resample)) / median(boot_resample)
}

Example 2: We don't know result

The the shape and spread of bootstrap distribution of \(\overline{X}\) will be similar to that of the sampling distribution.
